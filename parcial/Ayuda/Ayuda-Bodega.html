Mantenimiento de Bodegas
En esta forma se pueden realizar operaciones que afecte lo que es la tabla de desarrollo de carrera
se pueden realizar distintas operaciones. Por ejemplo, insertar nuevos empleados, modificarlos y
eliminarlos.
En esta forma se hace uso del navegador, un componente el cual permite realizar dichas operaciones
mencionadas anteriormente, las cuales lo permitir usar dependiendo los permisos que tenga el
usuario.
Ventana de Datos de bodegas:
Este mantenimiento tiene como fin el incluir uno de los requerimientos que era que fuera multibodega,
que nos permite almacenar diferentes bodegas en las cuales se hace el manejo del inventario.
Cuando se tenga todos los datos ya sea para modificarlos o ingresarlos, se tiene que usar el botn de
guardar si se desea guardar dentro de la tabla, si se desea cancelar, deber de presionar el botn de
cancelar.
Si se desea eliminar un elemento de la tabla deber de seleccionarlo al igual que con el botn de
modificar, con la diferencia que se deber de presionar el botn de eliminar
Lo que corresponde el botn de modificar, funciona
seleccionando un elemento dentro de lo que es la tabla
de desarrollo de carrera, para seleccionar un elemento
de la tabla se puede mover con los botones de inicio,
anterior, siguiente y final. Luego de haber seleccionado
uno se, puede modificarlos ingresando nuevos datos
donde corresponde en el formulario.
