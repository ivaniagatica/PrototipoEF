Ayuda Traslado de productos
Mantenimiento de Traslado
En esta forma se pueden realizar operaciones que afecte lo que es la tabla de desarrollo de carrera
se pueden realizar distintas operaciones. Por ejemplo, insertar nuevos empleados, modificarlos y
eliminarlos.
En esta forma se hace uso del navegador, un componente el cual permite realizar dichas operaciones
mencionadas anteriormente, las cuales lo permitirá usar dependiendo los permisos que tenga el
usuario.
Ventana de Datos de traslado:
En este mantenimiento es donde se controlan los traslados de productos, con sus fechas, la cantidad
de producto que se esta enviando y el tipo de producto que se esta trasladando, esto con el fin de un
mejor control de los traslados de productos.
Cuando se tenga todos los datos ya sea para modificarlos o ingresarlos, se tiene que usar el botón de
guardar si se desea guardar dentro de la tabla, si se desea cancelar, deberá de presionar el botón de
cancelar.
Si se desea eliminar un elemento de la tabla deberá de seleccionarlo al igual que con el botón de
modificar, con la diferencia que se deberá de presionar el botón de eliminar
Lo que corresponde el botón de modificar, funciona
seleccionando un elemento dentro de lo que es la tabla
de desarrollo de carrera, para seleccionar un elemento
de la tabla se puede mover con los botones de inicio,
anterior, siguiente y final. Luego de haber seleccionado
uno se, puede modificarlos ingresando nuevos datos
donde corresponde en el formulario.
